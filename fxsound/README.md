# FXSound presets

`.fac` preset files for the [FXSound](https://www.fxsound.com/) Windows audio enhancer.

## How to load

1. Open **FXSound**
2. Go to `Settings → Presets → Import`
3. Select a `.fac` file from this folder

## Presets

| File | Genre | Character |
|---|---|---|
| `Pop.fac` | Pop | Balanced vocals and synth-pop clarity |
| `rap.fac` | Rap / Hip Hop | Vocal mids and punchy bass |
| `Trapmetal.fac` | Trap Metal | Aggressive lows and scream separation |
| `lofi.fac` | Lo-Fi / Chill | Warm analog feel, rolled-off highs |

## Band layout (FXSound)

FXSound uses 10 EQ bands at fixed center frequencies:
62.5 Hz · 110 Hz · 230 Hz · 370 Hz · 683 Hz · 1279 Hz · 2150 Hz · 5300 Hz · 10000 Hz · 12000 Hz

These differ from the EARS 11-band layout — see `preset-map.json` at the root for genre cross-references between the two tools.
